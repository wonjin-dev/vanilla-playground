1. 텍스트를 입력받을 수 있는 input 요소 하나를 만들고 body에 추가
2. 텍스트를 입력하고 엔터키를 누르면, 입력한 값이 1이상 60이하 숫자인지 확인
3. 입력한 값이 1이상 60 이하 숫자가 아니라면 '숫자를 입력해주세요'란 콘솔 로그를 노출하고, 입력값을 초기화
4. 1에서 60까지의 숫자이면 span 요소와 'clear' 버튼 추가
5. span에 숫자값을 노출하고 타이머 시작
6. 타이머가 돌아가는 중간에 'clear' 버튼 클릭 시 타이머 취소
7. 타이머 취소 시 4번에서 추가힌 span요소와 'clear'버튼 삭제
